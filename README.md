# StockAutomation
I want to design an app to help people to manage their financial plan
A form is required

Flow: 
1: 問卷調查表單填寫 -> 使用google form, 以Line機器人作為主要和使用者溝通的介面
2: 將使用者填的資料存到SQL/DB -> mysql
3: API從資料庫撈數據，給AI分析數據 -> 將分析內容紀錄回SQL當作給使用者的投資策略
4: 給使用者回饋 -> 將聊天內容紀錄回SQL
5: 每天主動發送大盤資訊給使用者/每週主動追蹤使用者財務狀況
6: 額外功能：使用者如果想調整財務策略，可以和AI說
